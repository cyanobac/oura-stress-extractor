# Relax v1.0.0 — First Stable Release

This release publishes the first stable version of Relax, including its full
architecture, threat model, and governance kernel. The service is now ready for
public use and future evolution.

## Highlights
- Fully documented architecture and extraction pipeline.
- Hardened deployment behind Cloudflare and Caddy.
- Non-root containers with dropped capabilities.
- Bounded resource usage and rate limiting.
- Reproducible Docker-based deployment.
- Governance kernel defining architecture, threat model, and roadmap.
